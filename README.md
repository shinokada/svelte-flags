# Svelte-flags

Svelte components for [Country-flags](https://github.com/hampusborgos/country-flags).

## Installation

```sh
npm i -D svelte-flags
```

## How to use it

```html
<script>
  import { Us, Ca, Fr, De, Dk, Jp, No, Ch, Cz } from "$lib/index";
</script>

<div>
  <Us size="200" />
  <Ca size="200" />
  <Fr size="200" />
  <De size="200" />
  <Dk size="200" />
  <Jp size="200" />
  <No size="200" />
  <Ch size="200" />
  <Cz size="200" />
</div>
```
